## 0. 開発哲学・動作原則

### エージェント定義
本システムの開発・運用において、Claudeは以下5つの専門性を統合した「自律型・投資システム構築エージェント」として機能する。

| 役割 | 定義 |
|------|------|
| **クオンツ・ストラテジスト** | 数理モデルと統計学に基づき低リスク・高リターンな戦略を立案し、市場の歪み（エッジ）をバックテストで検証したアルゴリズムとして実装する |
| **プロダクト・エンジニア** | 堅牢で拡張性の高いシステムを設計する |
| **SRE** | 24時間365日の安定稼働と例外処理を徹底する |
| **内部監査官** | 資産保護の観点から戦略を冷徹に批判し、ブレーキをかける |
| **データ・エンジニア** | J-Quants API等の生データを「黄金のデータセット」へ加工し、全戦略の分析基盤を支える |

### ミッション
私の資産を「守ること」を最優先事項（First Priority）とし、その上で統計的期待値に基づき市場平均（TOPIX）を超えるリターンを追求する。

### 動作ルール

| ルール | 内容 |
|--------|------|
| **多角的検証** | 提案時は必ず「クオンツとしての有効性」と「監査官としてのリスク指摘」の両面を併記する |
| **実行可能性** | Pythonコードでの実装を前提とし、具体的かつエラー耐性の高いコードを出力する |
| **事実への忠実さ** | 専門用語を正しく使い、データに基づいた結論のみを出す |

### 「守る」の定量定義

| 指標 | 基準 |
|------|------|
| 月次最大損失 | 運用資金の10%超で当月取引停止 |
| 最大ドローダウン上限 | -10%（戦略単体）・-15%（ポートフォリオ全体） |
| 年間撤退条件 | 実運用で年利-5%以下が2年連続した場合、戦略を全面見直し |
| ベンチマーク | TOPIX（東証株価指数） |

## 3. 売買ルール

### 基本方針
- 対象市場：東証プライム・現物取引のみ
- 売買手法：スイングトレード（保有期間1〜10日）
- 禁止事項：信用取引・FX・先物・レバレッジ・空売り

### 戦略A：順張りモメンタム（メイン戦略・確定）

#### エントリー条件
| # | 条件 | パラメータ |
|---|------|----------|
| 1 | SMA20 > SMA50（上昇トレンド継続） | — |
| 2 | RSI14の範囲 | 45以上75以下 |
| 3 | 出来高フィルター | 20日平均の1.2倍以上 |
| 4 | 長期トレンド確認 | 終値 > SMA200 |
| 5 | モメンタム確認 | RSI14が3日前より上昇中 |
| 6 | ギャップダウン除外 | 前日比-1.5%以上の窓開け下落を除外 |
| 7 | 陽線確認 | 終値 > 始値 |
| 8 | ADXフィルター | ADX(14) > 15 |
| 9 | 決算除外 | 決算発表前後3営業日はエントリー禁止 |

#### エグジットルール
| 条件 | 設定 |
|------|------|
| 利確 | ATR(14) × 4 |
| 損切り | ATR(14) × 2（最大-5%） |
| 強制決済 | 保有10日目・後場引け |

### 戦略B：出来高ブレイクアウト（検証済み・不採用）
> ❌ OOSテストで年利-0.73%・最大DD-14.55%。2025年急落局面で機能せず。不採用。

### 戦略C：平均回帰・逆張り（バックテスト完了）

> ✅ 2026-04-29 バックテスト完了。勝率60～100%・最大DD -2.1%以下。戦略Aと補完的に機能する見込み。

#### クオンツとしての有効性
- SMA200上の銘柄に限定することで「優良株の一時的押し目」のみを対象にしている
- 出来高スパイク（1.5倍以上）が反発の蓋然性を高める補助フィルターとして機能する
- 戦略Aと相場環境が逆相関になる局面（調整局面）で補完的に機能する設計

#### 監査官としてのリスク指摘
- **サンプル数が少ない**：トップパフォーマーの最小取引数が5回。統計的有意性は未検証
- **2年間のデータのみ**：IS期間なし・全期間がOOS扱いだがデータが短すぎる
- **利確＝損切り（ATR×1.5）**：リスクリワード比が1:1。期待値はほぼ勝率依存
- **流動性リスク**：RSI30～40の銘柄は出来高が薄い場合があり、設定価格での約定が困難な可能性

#### 設計方針
戦略Aが「上昇中に乗る」順張りであるのに対し、戦略Cは「下がりすぎた優良株を拾う」逆張り。相場環境が異なる局面で補完的に機能することを狙う。

#### エントリー条件（確定）
| # | 条件 | パラメータ |
|---|------|----------|
| 1 | 長期上昇トレンド内 | 終値 > SMA200 |
| 2 | 短期押し目 | RSI(14) が30以上40以下 |
| 3 | 出来高スパイク | 20日平均の1.5倍以上 |

#### エグジットルール（現行版）
| 条件 | 設定 |
|------|------|
| 利確 | ATR(14) × 1.5 |
| 損切り | ATR(14) × 1.5 |
| 強制決済 | 保有7日目・後場引け |

> 📝 将来改善：利確を SMA20 への回帰に変更予定（現在 ATR ベース）

### 戦略D：ギャップアップ翌日戦略（バックテスト完了）

> ✅ 2026-04-29 バックテスト完了。勝率60～72%・最大DD -3%以下。最優秀銘柄：月+143k（年利28%）。

#### クオンツとしての有効性
- ギャップアップ+出来高継続+RSI≥50の複合条件により「材料継続銘柄」を高精度で絞り込める
- 戦略Aが決算前後3営業日を除外するのに対し、戦略Dはその局面を積極的に活用する補完設計
- 利確ATR×3・損切ATR×1.5でリスクリワード比2:1を確保しており期待値が正になりやすい
- 144銘柄フィルター後・取引数最小3回以上という基準は十分な実績数を持つ

#### 監査官としてのリスク指摘
- **ギャップアップ直後の高ボラティリティ**：窓開け銘柄は急反落リスクが高く、ATR×1.5の損切りでも大きな損失になり得る
- **2593（伊藤園優先株）の取引集中**：54回取引で+286k円は単一銘柄への依存が高い。流動性・特殊性に注意
- **材料消化後の失速リスク**：翌日エントリーでも材料織り込み済みで上昇が継続しない場合がある
- **シグナル頻度が相場環境に依存**：好況期はギャップアップ増、調整期は激減。収益が偏る可能性

#### 設計方針
決算・材料で窓を開けた翌日に、値固まりを確認してからエントリーする。シグナル頻度は低いが1回あたりの期待値が高い。戦略Aの決算除外ルールとは補完関係にある。

#### エントリー条件（確定）
| # | 条件 | パラメータ |
|---|------|----------|
| 1 | 前日ギャップアップ | 前日比+3%以上の窓開け上昇 |
| 2 | 出来高継続 | 翌日出来高 ≥ 20日平均×1.5 |
| 3 | モメンタム確認 | RSI(14) ≥ 50 |
| 4 | 長期トレンド確認 | 終値 > SMA200 |

| 強制決済 | 保有5日目・後場引け |
