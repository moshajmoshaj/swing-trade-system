# swing-trade-system

## 起動時の必須手順

会話が始まったら、**最初に必ず** `docs/swing_trade_master.md` を読み込んでください。
このファイルがプロジェクトの設計書・仕様書・作業履歴の中心です。

```
Read: docs/swing_trade_master.md
```

読み込んだ内容を踏まえて、ユーザーの質問や作業依頼に答えてください。

## Phase 4 PDCAログ記録ルール

作業完了時（会話を終える前）に、以下を確認して `logs/pdca_log.txt` に記録すること。

記録対象：
- 今回の作業で「変えたかったが Phase 4 凍結ルールにより保留した」パラメータ・ロジック
- 気づいたシステムの問題点・改善案で、今すぐ変更すべきでないもの
- OOS検証が未実施のまま変更したもの（前提乖離の記録）

記録形式：
```
[YYYY-MM-DD] 変えたかった内容 → 理由・保留の根拠
```

記録すべき内容がなければスキップしてよい。
ユーザーから指示がなくても自発的に記録する。
